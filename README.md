# receipt-split-manager

レシートの商品項目を家計簿用データとして整理するためのWebアプリです。高額品は購入日の実支出として記録しつつ、家計簿上の入力予定を数ヶ月に分けて作成・管理できます。

初期版は手動入力のみです。OCR、カメラ撮影、CSV出力、電子レシート取り込みはまだ実装していません。

## 主な機能

- 商品入力
  - 購入日
  - 店舗名
  - レシート上の商品名
  - 正式な商品名
  - 金額（税込）
  - 分類
  - 通常入力 / 分割入力
- 分割入力
  - 分割月数
  - 開始月
  - 月額入力額の自動計算
  - 端数は最終月で調整
  - メモ
- 今月の入力予定
  - 今月入力すべき分割項目を表示
  - 未入力合計と予定合計を表示
- 月別予定一覧
  - 月ごとの配分入力予定を表示
  - 入力済み / 未入力の切り替え
- 登録済み商品一覧
  - 実際に購入した商品の記録を表示
- 学習辞書
  - 過去に入力したレシート表記、正式名、分類、店舗名、金額帯、確定回数、最終使用日を保存
  - 同じレシート表記を入力したときに候補を表示

## データの考え方

このアプリでは、以下を別データとして扱います。

- 実支出: 実際に購入日に支払った金額
- 配分入力: 家計簿上で月ごとに割り当てる金額

家計簿上の入力額は税込金額を基準にしています。

## データ保存

データはブラウザの `localStorage` に保存されます。

外部サーバー、ログイン、クラウド同期は使用していません。個人情報や入力内容は外部送信されません。

`localStorage` 保存のため、以下の場合にデータが消える可能性があります。

- ブラウザを変更した場合
- 別の端末で開いた場合
- ブラウザのキャッシュやサイトデータを削除した場合
- プライベートブラウズなど保存が残らない環境で使った場合

## 起動方法

依存関係をインストールします。

```bash
npm install
```

開発サーバーを起動します。

```bash
npm run dev
```

ブラウザで以下を開きます。

```text
http://localhost:5173/
```

## ビルド方法

```bash
npm run build
```

ビルド結果は `dist/` に出力されます。
