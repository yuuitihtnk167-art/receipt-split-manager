# receipt-split-manager

家計簿の入力を月ごとに分けて管理するためのWebアプリです。

高額品や任意の商品を購入日の実支出として登録しつつ、家計簿上では数ヶ月に分けた「分割入力予定」として確認できます。

## 主な機能

- 商品入力
  - 購入日
  - 店舗名
  - 商品名
  - 正式な商品名
  - 金額（税込）
  - 分類
  - 通常入力 / 分割入力
- 分割入力
  - 分割月数
  - 開始月
  - 月額入力額の自動計算
  - 端数は最終月で調整
  - メモ
- 今月の入力予定
  - 今月入力すべき分割項目を表示
  - 未入力合計と予定合計を表示
  - 入力済み / 未入力の切り替え
- 月別予定一覧
  - 月ごとの配分入力予定を表示
  - 入力済み / 未入力の確認
- 登録済み商品一覧
  - 実際に購入した商品の記録を表示
  - 商品削除
- データ管理
  - JSONバックアップ / 復元
  - 商品一覧CSV / 分割予定CSVの出力
- GitHub Pages / PWA

## データの考え方

このアプリでは、以下を別データとして扱います。

- 実支出: 実際に購入日に支払った金額
- 配分入力: 家計簿上で月ごとに割り当てる金額

家計簿上の入力額は税込金額を基準にしています。

## データ保存

データはブラウザの `localStorage` に保存されます。

外部サーバー、ログイン、クラウド同期は使用していません。個人情報や入力内容は外部送信されません。

`localStorage` 保存のため、以下の場合にデータが消える可能性があります。

- ブラウザを変更した場合
- 別の端末で開いた場合
- ブラウザのキャッシュやサイトデータを削除した場合
- プライベートブラウズなど保存が残らない環境で使った場合

重要なデータは、データ管理タブからJSONバックアップしてください。

## 起動方法

依存関係をインストールします。

```bash
npm install
```

開発サーバーを起動します。

```bash
npm run dev
```

ブラウザで以下を開きます。

```text
http://localhost:5173/
```

スマホ実機から確認する場合は、同じWi-Fi内のPCでホストを指定して起動します。

```bash
npm run dev -- --host 0.0.0.0
```

PCのローカルIPアドレスを確認し、スマホのブラウザで以下のようにアクセスします。

```text
http://PCのIPアドレス:5173/
```

## ビルド方法

```bash
npm run build
```

ビルド結果は `dist/` に出力されます。
